Automatiserad provisionering av infrastruktur

Beskrivning: Automatiserad provisionering av infrastruktur så som mjukvarudefinierat nätverk, brandväggar, DNS, loggningssystem, rättighetsstyrning för användare m.m.

Input: Skript för infrastrukturprovisionering, "recept", manifest (ex. kubernetes yaml), playbooks (ex Ansible playbooks)

Output: Provisionerad infrastruktur

Verktygsbehov: Automationsverktyg

Output: Ny containerinstans
